# CV_Computer-Vision-Master-Degree
Repository containing educational resources, assignments, and projects related to Computer Vision courses pursued during a Master's Degree program. Focused on image processing techniques, deep learning models, and practical implementations in the field of computer vision.

**Лабораторная №1 — Анализ бинарных изображений классическими алгоритмами с использованием библиотек компьютерного зрения**
- Цель работы: получение навыков работы с библиотеками компьютерного зрения **OpenCV** и **dlib** для базовых задач обработки изображений.

**Лабораторная №2 — Сегментация изображений и выделение границ с использованием OpenCV**
- Исследование влияния параметров контрастной ограниченной адаптивной эквализации гистограммы (CLAHE) на качество последующей обработки изображений
- Сравнительный анализ эффективности детекторов границ Кэнни (Canny) и лапласиана гауссова фильтра (Laplacian of Gaussian)
- Оценка устойчивости детектора ключевых точек ORB к локальным изменениям яркости и контраста
